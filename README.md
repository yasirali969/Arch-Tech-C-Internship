# 🎲 C++ Dice Rolling Simulator

A simple console-based Dice Rolling Simulator built using C++.

This project generates two random dice values every time the user rolls the dice and allows continuous rolling until the user chooses to quit.

** Features **

Random dice generation using rand()

Simulates rolling 2 dice

User can roll multiple times

Simple and beginner-friendly C++ project

Uses loops, conditions, and random number generation

**  Technologies Used **

C++

iostream

ctime

cstdlib

**  Concepts Practiced **

Loops (do-while)

Conditional statements

Random number generation

User input/output

Basic game logic

** How It Works **

Program generates 2 random dice numbers between 1–6.

Displays both dice values.

Asks the user whether they want to roll again.

Continues until the user enters N.
