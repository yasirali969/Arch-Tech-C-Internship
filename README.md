# 🎲 C++ Dice Rolling Simulator

A simple console-based Dice Rolling Simulator built using C++.

This project generates two random dice values every time the user rolls the dice and allows continuous rolling until the user chooses to quit.

** Features **

Random dice generation using rand()

Simulates rolling 2 dice

User can roll multiple times

Simple and beginner-friendly C++ project

Uses loops, conditions, and random number generation

**  Technologies Used **

C++

iostream

ctime

cstdlib

**  Concepts Practiced **

Loops (do-while)

Conditional statements

Random number generation

User input/output

Basic game logic

** How It Works **

Program generates 2 random dice numbers between 1–6.

Displays both dice values.

Asks the user whether they want to roll again.

Continues until the user enters N.

# 📝C++ To-Do List Manager

A console-based To-Do List application built using C++ that helps users manage daily tasks efficiently.
This project allows users to add, view, complete, delete, save, and load tasks using file handling.

 **Features** 

1) Add new tasks

2) View all tasks

3) Mark tasks as completed

4) Delete tasks

5) Save tasks to a file

6) Load tasks automatically from file

7) Progress tracking system

8) Simple menu-driven interface

 **Technologies Used** 

C++

Vectors (vector)

File Handling (fstream)

String Streams (stringstream)

Functions & Structures

 **Concepts Practiced**
 
Object-like data management using struct

Dynamic storage using vector

File handling in C++

Functions and modular programming

Loops and condition handling

Data persistence

 **File Handling** 
Tasks are stored in a text file so that data remains saved even after closing the program.
